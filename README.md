# ToursApplication

To run server:

```
npm install
```
```
npm run start
```

in "server" folder

---

To run app:

```
npm install
```

```
npm run start
```

in "app" folder

---

Server running on localhost:8080

App running on localhost:4200

---

After initializing:

To run server in dev mode:

```
npm run watch 
```

```
npm run dev
```

in "server" folder

---

To run app in dev mode:

```
npm run dev
```

in "app" folder
